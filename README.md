# NoteBook (in development)
## Note taking application
- Users can create an account
- Save notes to their account
- Organise notes by the chapters they correspond with in a particular book

## Frontend
- React with Typescript

## Backend
- Springboot with RESTful CRUD endpoints
- Spring securit for user auth
- PostgreSQL db running in a Docker container
- Makes use of DTO to separate response data from enities
- Controller > Service > Repository pattern
